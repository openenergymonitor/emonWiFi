# emonWiFi

This small adapter allows the following OpenEnergyMonitor products to be used with a WiFi connection.

- [emonTx4](https://github.com/openenergymonitor/emontx4)
- [emonTx5](https://github.com/openenergymonitor/emontx5)
- [emonTx6](https://github.com/openenergymonitor/emon32)

## Setup

### Installing the pin headers

You need to fit the connectors that correspond to the device you are attaching it to. There are different headers for each of the boards listed above. The headers face downwards, away from the side with the USB-C socket.

### Installing software

To install firmware for the first time, hold down the button while plugging in the USB-C cable. This puts the ESP32 module into the bootloader mode, allowing firmware to be uploaded over the USB connection.
